## ✨ Key Features of The Project
- **🌍 Multi-language Support**: Seamless toggle between **English (US)** and **Portuguese (BR)**.
- **⚡ Performance First**: Built with **Vite** for near-instant load times.
- **🎨 Modern UI/UX**: Crafted with **shadcn/ui** and **Tailwind CSS**, ensuring a clean, accessible, and professional aesthetic.
- **📱 Fully Responsive**: Optimized for all devices, from mobile phones to high-resolution desktops.
- **☁️ Cloud Integrated**: Assets like my CV/Resume are securely stored in **Cloudflare R2** and the site is hosted on **Vercel**.

## 🛠️ My Tech Stack

## 📁 Project Structure
```text
src/
├── components/       # Reusable UI components and page sections
│   ├── sections/     # Modular sections (Hero, About, Projects, Experience)
│   └── ui/           # Base UI components (shadcn/ui)
├── contexts/         # Theme and Internationalization (i18n) logic
├── data/             # Centralized data (Projects, Experience, Skills, Translations)
├── hooks/            # Custom hooks for language and responsiveness
└── pages/            # Main application layouts
```
